# 周强 — 个人学术主页

南昌大学工程建设学院 周强教授的个人学术主页。

## 关于

周强，教授，博士生导师，江西省力学学会理事。长期从事土木工程与工程力学专业相关教学和科研工作，以村镇房屋抗震研究为特色方向。

## 发布

本网站为纯静态 HTML 页面，可直接部署到 **GitHub Pages**：

1. Fork 或上传本仓库到 GitHub
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支，根目录 `/ (root)`
4. 保存后即可通过 `https://<username>.github.io/<repo>/` 访问

## 本地预览

直接用浏览器打开 `index.html` 即可。
